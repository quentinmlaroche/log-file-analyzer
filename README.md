# Log File Analyzer

Simple Python tool that analyzes a log file and generates a summary of log levels.

## Features
- Reads a text-based log file
- Counts INFO, WARNING, and ERROR entries
- Displays a clear summary in the console

## Why this project
This project was created to practice building a simple, structured backend tool and working with file input, parsing, and basic data processing.

## Technologies
- Python 3

## How it will work
The tool will read a log file line by line, identify log levels, and produce a summarized report showing the number of occurrences of each level.

## Future improvements
- Support for configurable log levels
- Export results to a file
- Add basic automated tests
